안녕하세요.
저는 효율적이고 견고한 백엔드 시스템을 설계하고 개발하는 백엔드 개발자 이현승입니다.
새로운 기술 학습과 문제 해결에 항상 열정적이며, 최적화된 코드를 통해 사용자 경험(UX)을 간접적으로 개선하는 일에 집중합니다.

소개

저는 사용자의 경험(UX)이 프론트엔드뿐만 아니라 데이터 처리 및 응답 속도에서 시작된다고 믿는 백엔드 개발자입니다.
Spring Boot를 주력으로 다루며, 시스템의 안정성과 효율성을 최우선으로 고려하여 견고하고 확장 가능한 아키텍처를 구축하는 것을 즐깁니다.
또한, 효율적인 팀워크와 동료와의 긴밀한 협업이 프로젝트 성공의 핵심이라고 믿으며, 열린 소통을 중요시합니다.
새로운 기술 스택을 빠르게 학습하여 서비스에 적용하는 것에 열정을 가지고 있습니다.

AI 모델이 분석한 입찰 관련 데이터를 백엔드에서 받아 통계적으로 처리하고 시각화하여 입찰 전략 수립을 지원하는 시스템 구축. AI-백엔드 데이터 연동의 효율성과 다중 권한(유저, 관리자, 분석사) 제어가 핵심 목표였습니다.

[2] 주요 역할 및 기술 적용

  • AI 데이터 처리 및 통계: AI 팀이 MySQL에 저장한 원시 데이터를 Spring Boot에서 조회하여, 복잡한 통계 로직(분포, 예측치 등)을 구현하고 결과를 화면에 제공하는 API를 개발함.
  • 성능 최적화: 통계 데이터 조회 속도 향상을 위해 MyBatis를 활용한 SQL 쿼리 튜닝 및 MySQL 인덱싱 전략을 적용하여, 데이터 처리 성능을 최적화함.
  • 보안 및 권한 관리: JWT 기반의 인증 시스템을 구축하고, 사용자 역할(유저, 관리자, 분석사)에 따라 접근 가능한 통계 및 관리 기능에 대한 권한 제어 로직을 견고하게 구현함.
  • CI/CD 및 배포 자동화: Docker를 사용하여 애플리케이션을 컨테이너화하고, Jenkins를 통한 자동 빌드 및 AWS EC2 환경으로의 배포 파이프라인을 구축하여 개발/운영 환경의 일관성 및 배포 속도를 확보함.

[3] 결과 및 성과

AI 분석 결과에 기반한 통계 데이터 조회 API의 평균 응답 시간을 50ms 이내로 단축하여 사용자 경험을 개선함. Docker 및 Jenkins 기반 CI/CD 구축을 통해 배포 오류율을 낮추고, AI 모델 업데이트에 따른 백엔드 배포 시간을 단축하여 운영 효율성을 크게 향상시킴.

[1] 프로젝트 개요 및 목표

차량 발주 프로세스 전반을 관리하는 시스템을 Node.js 기반의 AWS 서버리스 아키텍처로 구축. 내부망 DB(Postgres/Oracle)와 외부망 API 간의 안전한 접근 경로를 확보하고, 웹 및 모바일 앱 사용자들의 복잡한 권한을 정확히 제어하는 것이 목표.

[2] 주요 역할 및 기술 적용

  • 내/외부망 연동 및 NAS 접근: 외부망에서 동작하는 API가 VPC 내 NAT Gateway를 통해 내부망 DB에 접근하도록 경로를 설계하고, SFTP 프로토콜을 사용하여 내부망 NAS의 파일을 안전하게 조회하는 로직을 구현함.
  • 서버리스 아키텍처 구축: AWS Lambda와 API Gateway를 활용하여 Node.js 기반의 서버리스 API를 개발하고, CloudWatch를 통한 모니터링 체계를 구축하여 시스템 운영 효율 및 확장성을 확보함.
  • 이중 인증/인가 설계: 웹 환경에서는 JWT 토큰 기반 로그인을, 모바일 앱 연동에서는 Cognito를 활용한 인증 방식을 도입하여, 사용 환경에 따른 이중 인증 및 권한 제어 로직을 설계함.
  • 외부 통신 모듈 통합: 차량 발주 및 승인 알림을 위해 Amazon SES를 통한 Mail 발송 모듈과 LG U+ 문자 발송 시스템을 연동하여 다채널 알림 시스템을 구축함.
  • 운영 인프라 관리: Route 53을 통해 도메인 및 Health Check를 관리하고, S3를 파일 스토리지로 연동하는 등 서버리스 시스템 전반의 AWS 인프라 구축 및 운영을 담당함.

[3] 결과 및 성과

AWS 서버리스 아키텍처 도입으로 인프라 관리 비용 및 운영 오버헤드를 절감함. NAT Gateway 및 VPC 설계를 통해 외부망 접근 시 내부 DB/NAS의 보안 요구사항을 완벽히 충족했으며, SFTP 연동으로 내부 파일 시스템 접근의 안정성을 보장함. JWT/Cognito 이중 인증 전략을 통해 견고하고 일관된 권한 체계를 확립하여 서비스 안정성을 높임.

  • 데이터 모델링 및 관리: 물질 특성, 기상 관측소 정보, 교통량 등 다양한 입력 데이터 구조를 설계하고, 데이터 입력/수정/조회 기능을 구현하여 계산에 필요한 데이터 관리 모듈을 완성함.
  • 결과 시각화 지원: 추출된 공용성 예측 값을 그래프 형태로 표현할 수 있도록 데이터를 처리하고, 가능 설계 여부를 판별하는 판정 로직을 구현하여 프론트엔드 연동을 지원함.

[3] 결과 및 성과

시스템을 설치형 프로그램으로 성공적으로 배포하여 네트워크 단절 환경에서도 중요 설계 예측 업무가 연속성을 갖도록 보장함. 구현된 공학적 계산식의 검증 결과가 기존 예상값과 100% 일치함을 확인하여 시스템의 최종 결과 정확성 및 신뢰도를 입증함. 복잡한 계산 로직을 안정적으로 구현하여, 기존 시스템 대비 설계 예측 및 타당성 판별 시간을 30% 단축시킴.

[2] 주요 역할 및 기술 적용
