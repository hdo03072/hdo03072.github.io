새로운 기술 학습과 문제 해결에 항상 열정적이며, 최적화된 코드를 통해 사용자 경험(UX)을 간접적으로 개선하는 일에 집중합니다.

소개

저는 사용자의 경험(UX)이 프론트엔드뿐만 아니라 데이터 처리 및 응답 속도에서 시작된다고 믿는 백엔드 개발자입니다.
Spring Boot를 주력으로 다루며, 시스템의 안정성과 효율성을 최우선으로 고려하여 견고하고 확장 가능한 아키텍처를 구축하는 것을 즐깁니다.
새로운 기술 스택을 빠르게 학습하여 서비스에 적용하는 것에 열정을 가지고 있습니다.

  • [협업/문서화] Swagger/OpenAPI를 도입하여 API 명세 자동화 환경을 구축하고, 프론트엔드 및 외부 협력팀과의 API 명세 커뮤니케이션 오류를 최소화

주요 프로젝트

[1] 프로젝트 개요 및 목표

일정 규모 이상의 사용자들에게 실시간으로 푸시 알림을 전달하는 백엔드 시스템 구축. **대규모 트래픽 처리 및 지연 시간 최소화**가 목표였습니다.

[2] 주요 역할 및 기술 적용

  • **Kafka 도입**: 알림 메시지 비동기 처리를 위해 메시지 큐를 도입하여 초당 1만 건 이상의 요청을 안정적으로 처리했습니다.
